# NYC Yellow Taxi Data Analysis 🚕📊

## Overview
Exploratory Data Analysis on NYC Yellow Taxi dataset to understand travel patterns, fare trends, and demand behavior.

## Dataset
Dataset sourced from NYC Taxi & Limousine Commission (TLC).
(Note: Dataset not included due to size)

## Key Insights
- Peak demand during evening hours
- Higher fare per mile on weekends
- Significant variation across boroughs

## Tools Used
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

## Learnings
- Data cleaning and preprocessing
- Handling large datasets efficiently
- Exploratory data analysis techniques
